# OGTTS Core

OGTTS Core is the core mod for Operation: Gregtech to Space. It provides the foundational content and systems required by the modpack.

## 🎯 Purpose

The goal of this mod is to supply essential items, machines, and systems while minimizing the need for external dependencies.
By centralizing core functionality here, the modpack becomes more stable, maintainable, and easier to expand.

Although primarily designed for the modpack, you are free to download and experiment with this mod on its own.

## 📦 Content & Design

This project is data- and story-driven. As a result:

- Some items or blocks may not have visible uses when the mod is used standalone
- Recipes and progression are defined within the modpack itself
- Full functionality is only experienced as part of Operation: Gregtech to Space

This is intentional and part of the design philosophy.

## ⚠️ Usage Outside the Modpack

If you use this mod independently:

- Expect incomplete progression or missing recipes
- Some features may appear unused or inaccessible

For the intended experience, use it within the official modpack.
